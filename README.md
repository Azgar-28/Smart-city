# Smart City App

Project scaffold for a Smart City application.

Folder structure:

```
smart-city-app/
├── README.md
├── frontend/
│   ├── package.json
│   ├── public/
│   │   └── index.html
│   └── src/
│       ├── index.js
│       ├── App.js
│       ├── services/
│       │   └── api.js
│       └── components/
│           ├── CityList.js
│           └── ReportForm.js
└── backend/
    ├── package.json
    ├── server.js
    ├── routes/
    │   ├── cities.js
    │   └── reports.js
    ├── data/
    │   └── db.json
    └── .env.example
```

Run instructions included in frontend/backend package.json scripts.
